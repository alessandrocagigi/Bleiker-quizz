## Neste steg

1. Bytt ut filer i `assets/`.
2. Oppdater navn og kontaktinfo i footer pa alle sider.
3. Legg inn egne quizsporsmal.
4. Last opp repo til GitHub.
5. Aktiver GitHub Pages:
   - GûË til `Settings` -> `Pages`
   - `Build and deployment`: `Deploy from a branch`
   - Velg branch `main` og mappe `/ (root)`
